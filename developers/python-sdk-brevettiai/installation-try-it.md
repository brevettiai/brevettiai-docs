# Installation - Try it

## Via git

Installing is criterion core via git is done with pip

```bash
pip install git+https://bitbucket.org/criterionai/core
```

{% hint style="info" %}
Running in a Jupyter Notebook? add a '!' in front of the commando to run a shell command
{% endhint %}
